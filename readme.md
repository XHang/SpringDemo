## 项目简介：率属于Spring示例项目，主要介绍SpringDataSource的使用
什么是SpringDataSource？
就是数据源啦，这个数据源对象可以交于Spring容器管理，所以又叫SpringDataSource
这个数据源能干嘛？
目前来说，他可以获取数据库连接对象，从而执行jdbc操作。
还可以实现连接池，避免资源浪费
